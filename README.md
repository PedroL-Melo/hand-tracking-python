# Hand Tracking em Tempo Real

Um script em Python que utiliza visão computacional para rastrear as 21 juntas da mão em tempo real usando a webcam.

## 🛠️ Tecnologias Utilizadas
* Python
* MediaPipe
* OpenCV

## 🚀 Como Executar
1. Clone este repositório.q
2. Instale as dependências rodando o comando: `pip install -r requirements.txt`
3. Execute o rastreador com o comando: `py main.py`

## Créditos
Este projeto utiliza o modelo de visão computacional MediaPipe, desenvolvido pelo Google e distribuído sob a licença Apache 2.0.
